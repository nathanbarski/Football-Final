NFL Stadiums Map Project

An interactive map of every current NFL stadium in the United States. Click any marker to see who plays there and key facts pulled from live Wikipedia data.

Open the Map

1. Objective

Create a website that displays a U.S. map with all NFL stadiums marked. Users can click each stadium to view quick facts.

2. Web Scraping

Source: Wikipedia stadium list. Tools: requests, BeautifulSoup, pandas, geopy.

import requests
from bs4 import BeautifulSoup
import pandas as pd

url = "https://en.wikipedia.org/wiki/List_of_current_NFL_stadiums"
headers = {
    "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/91.0.4472.124 Safari/537.36"
}
response = requests.get(url, headers=headers)

3. Database

Columns: Name, Capacity, Location, Surface, Roof, Opened (plus image and coordinates). Exported CSV: nfl_stadiums.csv.

Example query:

SELECT Name, Capacity, Location, Surface, Roof, Opened
FROM Stadiums;

4. Web Application

Interactive Folium map of the U.S. with markers for every stadium. Clicking a marker reveals the team, location, capacity, surface, roof type, year opened, and an image pulled from Wikipedia.
